## Einleitung

Mit Aktions-Kategorien können Aktionen in Kategorien gruppiert werden, welche dann in der Aktionsübersicht einer
Prozess-Instanz angezeigt werden.

![Konfiguration](/img/docs/de/config-action-categories.png)

Aktions-Kategorien werden unterhalb der Prozess-Situation in einer Prozess-Instanz angezeigt **(1)**. Standardgemäß
werden nur die aktiven Aktionen angezeigt. Über den Schalter "Inaktive Aktionen anzeigen" können die inaktiven Aktionen
eingeblendet werden.

Aktions-Kategorien werden im Konfigurationsbereich unter "Aktions-Kategorien" konfiguriert.

![Konfiguration](/img/docs/de/action-categories/config-action-categories-config.png)

Eine neue Kategorie wird über den Button "Kategorie anlegen" erstellt **(1)**. Mit den Pfeilen **(2)** kann die
Reihenfolge der Aktionen in der Kategorie geändert werden. Mit Auswahlfeld **(3)** kann eine Aktion in eine andere
Kategorie verschoben werden. Jeder Prozess hat standardgemäß eine "Versteckt" Kategorie **(4)**. Aktionen in dieser
Kategorie werden nicht auf der Benutzeroberfläche in der Allisa Plattform angezeigt. Mit den Pfeilen an der
Kategorie **(5)** kann die Reihenfolge der Kategorien angepasst werden.

<a name="create-category"></a>

## Kategorie anlegen

Mit dem "Kategorie anlegen"-Button öffnet sich ein Dialogfenster. Es kann ein Name, eine Beschreibung, ein Icon und eine
Farbe gewählt werden. Mit dem "Versteckt?"-Schalter wird festgelegt, ob diese Kategorie auf der Weboberfläche angezeigt
werden soll.

![Konfiguration](/img/docs/de/action-categories/config-action-categories-create.png)
